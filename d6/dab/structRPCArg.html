NO

OMITTED_NAMED_ARG 

Optional arg that is a named argument and has a default value of null. When possible, the default value should be specified.

OMITTED 

Optional argument with default value omitted because they are implicitly clear. That is, elements in an array or object may not exist by default. When possible, the default value should be specified.
