2 Inserts a copyright header for "The Bitcoin Core developers" at the top of the
3 file in either Python or C++ style as determined by the file extension. If the
4 file is a Python file and it has a '#!' starting the first line, the header is
5 inserted in the line below it.
6 
7 The copyright dates will be set to be:
8 
9 "<year_introduced>-<current_year>"
10 
11 where <year_introduced> is according to the 'git log' history. If
12 <year_introduced> is equal to <current_year>, the date will be set to be:
13 
14 "<current_year>"
15 
16 If the file already has a copyright for "The Bitcoin Core developers", the
17 script will exit.
18 
19 Usage:
20  $ ./copyright_header.py insert <file>
