Bitcoin
Benchmarking

Bitcoin Core has an internal benchmarking framework, with benchmarks for cryptographic algorithms (e.g. SHA1, SHA256, SHA512, RIPEMD160), as well as the rolling bloom filter.

Running

After compiling bitcoin-core, the benchmarks can be run with: 

src/bench/bench_bitcoin

The output will look similar to:
